    • forEach

      遍历数组

      var arr = ["a1", "b1", "c1"];
arr.forEach(function(item, index, array){
    console.log(item); // "a1" "b1" "c1"
    console.log(index); // 0 1 2
    console.log(array); // ["a1", "b1", "c1"]
});

    • includes

      用来判断一个数组是否包含一个指定的值 返回Boolean值

      // 存在返回true，不存在返回false
var arr = ["a1", "b1", "c1"];
var index1 = arr.includes('b1'); 
console.log(index1); // true
var index2 = arr.includes('d1'); 
console.log(index2); // false

    • indexOf

      查看某个元素在数组中的位置

      // 返回元素的下标，不存在返回-1
var arr = ["a1", "b1", "c1"];
var index1 = arr.indexOf('b1'); 
console.log(index1); // 1
var index2 = arr.indexOf('d1'); 
console.log(index2); // -1

    • join

      把数组的所有元素放入一个字符串。元素通过指定的分隔符进行分割

    • keys

    • lastIndexOf

    • map

    • pop

      删除并返回数组的最后一个元素

      var arr = ["a1", "b1", "c1"];
arr.pop();
console.log(arr); // ["a1", "b1"]

    • push

      向数组的末尾添加一个或者更多元素，并返回新的长度

      var arr = ["a1", "b1"];
arr.push('c1');
console.log(arr); // ["a1", "b1", "c1"]

    • reduce

    • reduceRight

    • reverse

      颠倒数组中元素的顺序

    • shift

      删除并返回数组的第一个元素

      var arr = ["a1", "b1", "c1"];
arr.shift();
console.log(arr); // ["b1", "c1"]

    • slice

      从某个已有的数组返回选定的元素

      • slice( start, end)
      • 返回一个新的数组 包含从start 到 end (不包含该元素)
      • 不改变原数组
      var arr = ["a1", "b1", "c1"];
var copyArr = arr.slice(); // 复制数组
console.log(copyArr); // ["a1", "b1", "c1"]
var sliceArr = arr.slice(0, 1); // 获取数组
console.log(sliceArr); // ["a1"]

    • unshift

      向数组的开头添加一个或更多元素，并返回新的长度

arr.unshift('c1');
console.log(arr); // ["c1", "a1", "b1"]
